# 书籍数据录入模板

本文件用于维护「詹青云推荐书籍大全」的书籍档案。正式录入前，建议先在 Excel、飞书表格或 Notion 中整理，再转换为 `src/data/books.ts` 中的对象。

## 字段说明

| 字段 | 类型 | 填写说明 | 可选值或示例 |
| --- | --- | --- | --- |
| `id` | 数字 | 唯一编号，不可与已有书籍重复 | `13` |
| `title` | 文本 | 书名 | `示例书名` |
| `author` | 文本 | 作者；多人可用 ` / ` 分隔 | `示例作者` |
| `cover` | 文本 | 封面路径；没有封面时留空 | `/covers/13.jpg` 或空字符串 |
| `themes` | 文本数组 | 书籍讨论的具体主题 | `公共表达、个人选择` |
| `tags` | 文本数组 | 用于搜索和归类的短标签 | `法律、辩论、自由` |
| `sourceType` | 枚举 | 与詹青云的关联类型 | `明确推荐`、`公开提到`、`引用讨论`、`主题延伸` |
| `sourceTitle` | 文本 | 节目、访谈、文章或书单名称 | `主题延伸书单` |
| `sourceUrl` | 文本 | 可核验的公开来源链接；未知时留空 | `https://example.com/source` |
| `evidence` | 文本 | 说明为何收录，以及可核验的具体依据 | `节目中明确提到书名，待补时间点` |
| `readingValue` | 文本 | 读者可以从书中获得什么 | `理解公共表达中的论证与倾听。` |
| `verifyStatus` | 枚举 | 来源是否已经人工核验 | `已核验`、`待核验` |
| `difficulty` | 枚举 | 阅读难度 | `入门`、`进阶`、`挑战` |
| `status` | 枚举 | 个人阅读进度 | `想读`、`在读`、`读过`、`暂不确定` |
| `notes` | 文本 | 个人备注或补充说明 | `适合与相关访谈一起阅读。` |

当前页面还使用以下辅助字段，它们也需要写入 `books.ts`：

| 字段 | 类型 | 填写说明 |
| --- | --- | --- |
| `topic` | 枚举 | 大类：`文学`、`历史`、`社会`、`哲学` |
| `year` | 数字 | 首次出版年份；公元前使用负数 |
| `coverColor` | 文本 | 无真实封面时的备用颜色，如 `#315e4d` |

## 表格模板

下面是制表符分隔的模板。复制代码块内容并粘贴到 Excel、飞书表格或 Notion，通常会自动拆分为列。

```tsv
id	title	author	cover	themes	tags	sourceType	sourceTitle	sourceUrl	evidence	readingValue	verifyStatus	difficulty	status	notes	topic	year	coverColor
```

数组字段 `themes` 和 `tags` 在表格中使用中文逗号 `、` 分隔。每格只填写纯文本，不要添加 JavaScript 引号或方括号。

## 完整示例

以下仅演示录入格式，不代表已经确认的詹青云推荐记录。

```tsv
13	示例书名	示例作者	/covers/13.jpg	公共表达、个人选择	辩论、自由、沟通	主题延伸	主题延伸书单		根据詹青云常讨论的公共表达、文学、社会议题延伸收录，待进一步核验	帮助读者理解公共讨论中的论证、倾听与选择。	待核验	入门	暂不确定	示例条目，确认真实来源后再更新。	社会	2020	#6f8068
```

对应的 `books.ts` 数据如下：

```ts
{
  id: 13,
  title: '示例书名',
  author: '示例作者',
  cover: '/covers/13.jpg',
  themes: ['公共表达', '个人选择'],
  tags: ['辩论', '自由', '沟通'],
  sourceType: '主题延伸',
  sourceTitle: '主题延伸书单',
  sourceUrl: '',
  evidence:
    '根据詹青云常讨论的公共表达、文学、社会议题延伸收录，待进一步核验',
  readingValue: '帮助读者理解公共讨论中的论证、倾听与选择。',
  verifyStatus: '待核验',
  difficulty: '入门',
  status: '暂不确定',
  notes: '示例条目，确认真实来源后再更新。',
  topic: '社会',
  year: 2020,
  coverColor: '#6f8068',
},
```

## 转换为 books.ts

1. 在表格中完成一行书籍数据，并检查 `id` 是否唯一。
2. 在 `src/data/books.ts` 的 `books` 数组末尾复制一条现有对象。
3. 按表格内容逐项替换字段值。
4. 将 `themes` 和 `tags` 按 `、` 拆分为字符串数组，例如：

```ts
themes: ['公共表达', '个人选择'],
tags: ['辩论', '自由', '沟通'],
```

5. 文本使用单引号包裹。文本中如有单引号，需要写成 `\'`。
6. 数字字段 `id` 和 `year` 不加引号；其他字段均按类型填写。
7. 没有封面或来源链接时使用空字符串：`cover: ''`、`sourceUrl: ''`。
8. 没有可靠来源时，不要使用 `明确推荐`。应填写 `主题延伸` 和 `待核验`。
9. 保存后运行 `npm run build` 和 `npm run lint`，确认字段与枚举值正确。

## 来源可信度

- `明确推荐`：有可访问的一手来源，能够确认推荐语境。
- `公开提到`：在公开内容中提到书名，但没有明确表达推荐。
- `引用讨论`：引用了书中观点、人物或内容展开讨论。
- `主题延伸`：根据相关议题补充的阅读线索，不代表詹青云直接提到。
- `已核验`：已经检查原始内容，来源标题、链接和语境相符。
- `待核验`：尚未找到或尚未完整检查一手来源。

可信度类型和核验状态必须分别填写。例如，一条记录可以是 `公开提到 + 已核验`，也可以是 `主题延伸 + 待核验`。
